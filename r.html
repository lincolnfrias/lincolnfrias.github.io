R



Para experimentar o R, sem instalar nada, clique aqui.


Instalação


São dois passos: instalar a linguagem e instalar a interface (RStudio).

Instalando a linguagem:

  • Clique aqui e selecione "Download R". Quando o download terminar, basta clicar duas vezes no arquivo.

Instalando a interface:

  • Clique aqui para baixar o instalador do RStudio (o Windows é o primeiro da lista).


Dicas


  • tutorial de R - uma apostila bem prática, mostrando como fazer diversas tarefas estatísticas no R.
  • curso-r.com: ótimos tutoriais em português
  • r-statistics.com: introdução à estatística em R (em inglês).
  • veja aqui uma comparação entre python e R.
  • comandos básicos de R - lista dos comandos mais utilizados.
  • livro Análise de dados com R
  • awesome-R: lista dos principais pacotes e materiais sobre R.
  • Datacamp: ótimo curso gratuito de R para iniciantes. Em inglês, mas não deixe de tentar.
  • CRAN (The Comprehensive R Archive Network) - a página oficial do R, onde há uma lista dos mais de dez mil pacotes (as extensões do R para tarefas específicas) e as task views (roteiros sobre áreas específicas, p. ex., econometria, análise multivariada etc.).
  • Stackoverflow - fórum muito sério de perguntas e respostas, onde tanto os usuários avançados quanto novatos de qualquer linguagem de programação tiram suas dúvidas.
  • Rbitrary: por que <- ao invés de =? Por que na.rm ao invés de na_rm? Explicações rápidas para algumas esquisitices do R.
